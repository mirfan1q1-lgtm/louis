## ğŸ“ Tugas (Assignments)

### Melihat Tugas

#### Akses Menu Tugas:
1. Dari dashboard, klik **"Lihat Tugas"** atau **"Kelas Saya"**
2. Pada halaman kelas, pilih tab **"Tugas"**

#### Daftar Tugas:
Tugas ditampilkan dalam tabel dengan informasi:
- **Judul Tugas**: Nama tugas
- **Tipe**: Wajib atau Tambahan
- **Deadline**: Tanggal dan waktu pengumpulan
- **Nilai**: Nilai yang diperoleh (jika sudah dinilai)
- **Status**: Status pengumpulan tugas

#### Status Tugas:
- **Belum**: Tugas belum dikumpulkan
- **Dikumpulkan**: Tugas sudah dikumpulkan, menunggu penilaian
- **Dinilai**: Tugas sudah dinilai guru
- **Terlambat**: Deadline sudah lewat, tugas belum dikumpulkan

### Mengumpulkan Tugas

#### Langkah-langkah Pengumpulan:

1. **Pilih Tugas**:
   - Klik **"Lihat"** pada tugas yang ingin dikumpulkan
   - Atau klik **"Kumpulkan"** jika tugas belum dikumpulkan

2. **Baca Instruksi**:
   - **Deskripsi Tugas**: Baca penjelasan tugas dengan teliti
   - **Deadline**: Perhatikan batas waktu pengumpulan
   - **Link Drive**: Jika ada, klik untuk melihat materi tambahan dari guru
   - **Total Poin**: Skor maksimal yang dapat diperoleh

3. **Siapkan File Tugas**:
   - Kerjakan tugas sesuai instruksi
   - Upload file ke Google Drive Anda
   - **Penting**: Set permission file ke **"Anyone with link can view"**
   - Copy link Google Drive

4. **Isi Form Pengumpulan**:
   - **Link Google Drive**: Masukkan link file tugas (wajib)
     - Format yang benar: 
       - `https://drive.google.com/file/d/...`
       - `https://docs.google.com/document/d/...`
       - `https://docs.google.com/spreadsheets/d/...`
   - **Jawaban Teks**: Tambahkan penjelasan atau jawaban tertulis (opsional)
   - Pastikan link dapat diakses sebelum mengumpulkan

5. **Kirim Tugas**:
   - Klik **"Kumpulkan Tugas"**
   - Tunggu konfirmasi berhasil
   - Status akan berubah menjadi "Dikumpulkan"
   - Notifikasi akan dikirim ke guru

#### Tips Pengumpulan Tugas:
- **Format Link**: Pastikan link Google Drive valid dan dapat diakses
- **Permission**: Set file Google Drive ke "Anyone with link can view"
- **Deadline**: Kumpulkan sebelum deadline untuk menghindari penalti
- **Backup**: Simpan salinan file di device Anda sebagai backup
- **Test Link**: Pastikan link dapat dibuka di browser incognito sebelum mengumpulkan

### Membatalkan Pengumpulan

Jika Anda sudah mengumpulkan tugas tetapi ingin mengubah file:

1. Pada tabel tugas, klik **"Batal"** pada tugas yang sudah dikumpulkan
2. Konfirmasi pembatalan
3. Sistem akan membatalkan pengumpulan sebelumnya
4. Status akan kembali ke "Belum"
5. Anda dapat mengumpulkan ulang dengan file yang baru

**Catatan**: 
- Hanya tugas yang sudah dikumpulkan tetapi belum dinilai yang dapat dibatalkan
- Setelah dinilai, tidak dapat membatalkan pengumpulan

### Melihat Nilai dan Feedback

1. **Akses Tugas yang Sudah Dinilai**:
   - Tugas yang sudah dinilai akan menampilkan nilai di tabel
   - Status akan berubah menjadi "Dinilai"

2. **Lihat Detail Penilaian**:
   - Klik **"Lihat"** pada tugas yang sudah dinilai
   - Lihat nilai yang diberikan
   - Baca feedback dari guru (jika ada)
   - Review komentar untuk perbaikan di masa depan

## ğŸ“¢ Newsroom (Berita & Pengumuman)

### Melihat Berita dan Pengumuman

#### Akses Newsroom:
1. Dari sidebar, klik **"Newsroom"**
2. Atau dari dashboard, lihat preview berita terbaru

#### Tampilan Newsroom:
- **Grid Layout**: Berita ditampilkan dalam format kartu
- **Lazy Loading**: Scroll ke bawah untuk memuat lebih banyak berita
- **Filter**: Filter berdasarkan tipe (Berita/Pengumuman)

#### Informasi Postingan:
- **Judul**: Judul postingan
- **Gambar**: Gambar utama (jika ada)
- **Excerpt**: Ringkasan singkat
- **Tanggal**: Kapan postingan dipublish
- **Prioritas**: Badge prioritas (Normal/Penting/Urgent)
- **Tipe**: Badge tipe (Berita/Pengumuman)

### Membaca Postingan Lengkap

1. **Klik Postingan**:
   - Klik pada kartu postingan untuk melihat detail lengkap

2. **Informasi Lengkap**:
   - **Judul**: Judul lengkap
   - **Gambar**: Gambar utama (jika ada)
   - **Excerpt**: Ringkasan (jika ada)
   - **Konten**: Isi lengkap postingan
   - **Tanggal Publish**: Kapan dipublish
   - **Pembuat**: Nama guru yang membuat postingan

3. **Komentar Disqus**:

   - Anda dapat menulis komentar atau membalas komentar lain
   - Komentar hanya muncul untuk postingan yang sudah dipublish
   - **Catatan**: Login ke Disqus diperlukan untuk berkomentar

### Tips Menggunakan Newsroom:
- **Cek Rutin**: Cek newsroom setiap hari untuk pengumuman terbaru
- **Prioritas**: Perhatikan postingan dengan badge "Penting" atau "Urgent"
- **Komentar**: Gunakan fitur komentar untuk bertanya atau berdiskusi
- **Share**: Bagikan postingan penting ke teman sekelas jika diperlukan

## ğŸ”§ Troubleshooting

### Masalah Umum dan Solusi:

#### Tidak Bisa Login:
- **Solusi**: 
  - Pastikan email dan password benar
  - Perhatikan huruf besar/kecil
  - Hubungi guru jika lupa password
  - Clear cache browser dan coba lagi

#### Tugas Tidak Bisa Dikumpulkan:
- **Solusi**: 
  - Pastikan link Google Drive valid
  - Pastikan file dapat diakses publik
  - Test link di browser incognito
  - Pastikan belum melewati deadline
  - Refresh halaman dan coba lagi

#### File Tidak Bisa Didownload:
- **Solusi**: 
  - Periksa koneksi internet
  - Coba refresh halaman
  - Coba browser lain
  - Hubungi guru jika masalah berlanjut

#### Notifikasi Tidak Muncul:
- **Solusi**: 
  - Refresh halaman
  - Clear cache browser
  - Logout dan login kembali
  - Periksa koneksi internet

#### Data Tidak Update:
- **Solusi**: 
  - Refresh halaman
  - Logout lalu login kembali
  - Clear cache browser
  - Periksa koneksi internet

#### Komentar Disqus Tidak Muncul:
- **Solusi**: 
  - Pastikan postingan sudah dipublish
  - Periksa koneksi internet
  - Pastikan JavaScript enabled
  - Coba browser lain

#### Link Google Drive Tidak Bisa Diakses:
- **Solusi**: 
  - Pastikan permission file set ke "Anyone with link"
  - Test link di browser incognito
  - Pastikan file tidak dihapus
  - Hubungi guru jika masalah berlanjut

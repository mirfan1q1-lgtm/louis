# LOUIS - Learning Operation Unified Interactive System

LOUIS adalah platform Learning Management System (LMS) modern yang dirancang untuk memudahkan proses pembelajaran digital. Sistem ini menyediakan antarmuka terpadu untuk guru dan siswa dalam mengelola kelas, tugas, absensi, dan komunikasi pembelajaran.

## Fitur Utama

- **Dashboard Interaktif**: Dashboard terpisah untuk guru dan siswa dengan statistik real-time
- **Manajemen Kelas**: Buat dan kelola kelas dengan mudah
- **Sistem Tugas**: Buat, distribusikan, dan nilai tugas secara digital
- **Absensi Digital**: Sistem absensi terintegrasi
- **Newsroom**: Publikasikan berita dan pengumuman penting
- **Leaderboard**: Pantau peringkat dan performa siswa
- **Notifikasi Real-time**: Tetap update dengan notifikasi penting

## Teknologi

- **Frontend**: React 18 + TypeScript
- **UI Framework**: Mantine UI
- **Routing**: React Router v7
- **Backend**: Supabase
- **Build Tool**: Vite
- **Styling**: Tailwind CSS

## SEO Optimized

LOUIS telah dioptimalkan untuk SEO dengan:
- Meta tags lengkap (Open Graph, Twitter Cards)
- Dynamic page titles dan descriptions
- Sitemap.xml dan robots.txt
- Structured data untuk search engines
- Mobile-friendly dan responsive design

## Instalasi

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
